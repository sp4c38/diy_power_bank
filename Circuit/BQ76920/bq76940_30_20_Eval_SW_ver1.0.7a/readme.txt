bqMAXIMO Windows installation software for bq769X0 AFEs


1. This program executes on Windows XP, Vista, and 7 32 or 64bit operating systems.  This installer assumes that a working EV2300 or EV2400 communications adapter with drivers is already installed on system.

2. Execute setup.exe.  This installs bqMAXIMO and all support components in c:\program files\Texas Instruments\bqMAXIMO directory.  A bqMAXIMO shortcut is placed on the desktop.  This shortcut can be used to execute the program after installation.  Other shortcuts are installed under "All Programs" "Texas Instruments".
